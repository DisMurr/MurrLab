MurrLab – AI Voice Platform

This Cloudflare Pages site hosts a static landing page. The live app (Streamlit UI and FastAPI API) runs as Python services and cannot execute on Pages. See deployment options below.

Local quick start

UI (Streamlit)
  • pip install -e .[web,api]
  • streamlit run run_ui_streamlit.py
  • Opens at http://localhost:8501
API (FastAPI)
  • uvicorn apps.api.advanced_voice_api:app --reload
  • Docs at http://localhost:8000/docs
Weights
  • Place model files in weights/ or set MURR_WEIGHTS_DIR
  • Or authenticate to the Hugging Face repo for automatic download

Deploy options

Cloudflare Tunnel
  • Expose your local UI (8501) or API (8000) under your domain
  • cloudflared tunnel run & route dns
Compute hosting
  • Run the Python services on a VM or PaaS (Render/Railway/Fly/Cloud Run)
  • Then point your domain via Cloudflare DNS/Proxy
Pages + external backend
  • Host a JS front-end on Pages; call your FastAPI deployed elsewhere

Configure Cloudflare Pages

  • Framework preset: None
  • Build command: (leave empty)
  • Build output directory: public
  • Root directory: /

Pages will publish this static site from the /public folder.
